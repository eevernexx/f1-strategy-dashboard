# F1 Strategy Intelligence — Dashboard (web)

A Finrise-style season dashboard for Formula 1, built with **Next.js (App Router) + Tailwind CSS + Framer Motion**. Dark card layout, F1-red accent, smooth on-load and interaction animations.

Real F1 data (2022–2024) is exported once from the Python pipeline to static JSON in `public/data/`, so the app is fully static and deploys to Vercel with no backend.

## Local development

```bash
cd web
npm install
npm run dev        # http://localhost:3000
```

## Refreshing the data

Data is generated by the Python exporter at the repo root (uses `fastf1.ergast`, HTTP only):

```bash
# from the repo root, with the project venv active
./venv/Scripts/python.exe export_data.py
```

This writes `web/public/data/<year>.json` and `web/public/data/index.json`.

## Deploy to Vercel

1. Push the repository to GitHub.
2. In Vercel, **New Project → Import** the repo.
3. Set **Root Directory** to `web` (the Next.js app lives in this subfolder).
4. Framework preset: **Next.js** (auto-detected). Build command `next build`, output handled automatically.
5. Deploy. No environment variables are required.

CLI alternative:

```bash
cd web
npx vercel        # follow prompts; confirm root is the web/ folder
npx vercel --prod
```

## What maps to what

| Finrise widget        | F1 content                                            |
| --------------------- | ----------------------------------------------------- |
| Total balance         | Championship leader points + cumulative area chart    |
| Top spending (rings)  | Constructor points share                              |
| Cards                 | Featured driver card (number, points, wins/poles)     |
| Budget (bars)         | Points standings — drivers / teams toggle             |
| Sales heatmap         | Per-driver per-round points heatmap                   |
| Transaction history   | Recent race results (Win / Podium / DNF + points)     |
| Quick transfer        | Title contenders (driver avatars)                     |
| Stats + line          | Reliability & momentum (rounds, finish rate, DNFs)    |
